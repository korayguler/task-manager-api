# Task Manager REST API

## Clone this repo

```bash
# Clone this repository
$ git clone https://github.com/korayguler/task-manager-api

# Go into the repository
$ cd task-manager-api
```

## Project setup

```bash
npm install

or

yarn

```

### Compiles and hot-reloads for development

```
npm run dev

or

yarn dev

```

### Compiles and minifies for production

```
npm run build

or

yarn build

```
